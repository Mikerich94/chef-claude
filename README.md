# Chef Claude 🍝

A React app that suggests recipes based on ingredients you have on hand, powered by the Anthropic Claude API.

## Tech Stack

- React
- Vite
- Anthropic Claude API

## Running Locally

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file and add your `VITE_ANTHROPIC_API_KEY`
4. Run `npm run dev`
